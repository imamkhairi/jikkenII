2.　基本機能 2



1. 箇条書き環境
  1. 記号を番号にする． enumerate 環境
  2. 記号を任意文字列にする． description 環境 (\item[文字列])
  3. 裏ワザ：各項目の間隔の調整． \setlength{\itemsep}{-3pt}
2. 文字飾りとパッケージ
  1. 文章を四角枠でかこむ． \fbox{文章} を使用．
  2. 文章を丸枠でかこむ． \ovalbox{文章} を使用．プリアンブル部に \usepackage{fancybox} を記述．
  3. 文章を二重枠でかこむ． \doublebox{文章} を使用．プリアンブル部に \usepackage{fancybox} を記述．
  4. 文章を影付き枠でかこむ． shadebox環境 の中に文章を記述する．プリアンブル部に \usepackage{ascmac} を記述．
  5. 文章をタイトル付き枠でかこむ．itembox環境 の中に文章を記述する．タイトルは{ }で囲む．プリアンブル部に \usepackage{ascmac} を記述．
3. 空領域
  1. \vspace{20mm}　縦方向に 20mm のスペースを挿入．
  2. \hspace{20mm}　横方向に 20mm のスペースを挿入．
4. 資料作成

　与えられた資料 (kushiro.pdf) を作成せよ．
